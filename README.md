A single small web app used as the shared subject for two NEBo tasks:

1. **Manage environments using built-in OS tools** (CLI / environment task)
2. **Use version control tools for development** (Git task)

## Structure

```
.
├── app/            # the simple web app (Node + Express)
├── scripts/        # OS shell scripts (windows/ + linux/)
├── node-scripts/   # NodeJS equivalents of the shell scripts
├── .githooks/      # git hooks
└── docs/           # task narratives, command logs, and the remote-setup guide
```
